# Odontoceti

[![rustc](https://img.shields.io/badge/rustc-1.78+-blue?style=flat-square&logo=rust)](https://www.rust-lang.org)
[![license](https://img.shields.io/badge/license-Apache-blue.svg?style=flat-square)](LICENSE)

The code in this branch is a prototype of Mysticeti. It supplements the paper [BlueBottle: Fast and Robust Blockchains through Subsystem Specialization]([https://arxiv.org/abs/2310.14821](https://sonnino.com/papers/bluebottle.pdf)) enabling reproducible results. There are no plans to maintain this branch.
